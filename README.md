## 💫 About Me

I'm a student developer with a strong interest in **AI-driven applications** and modern web technologies 🤖. I enjoy designing and building end-to-end systems — from clean user interfaces to backend logic and model integration.

Right now, I'm deep into **Agentic AI** — exploring how autonomous, goal-driven agents can reason, plan, and take action with minimal human intervention. Think LLM-powered agents, multi-agent orchestration, and tool-use pipelines. The future is agents that code themselves ✨.

I have hands-on experience with full-stack development and applied machine learning through internships and team projects. I like working on problems related to **automation, intelligence, and real-world product development**.

> Always curious, always building, and constantly learning 🚀
